Glossários

HTML
HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas na web. Ela utiliza uma série de elementos e tags para definir o conteúdo, como textos, imagens, links, tabelas e formulários. O HTML é interpretado pelos navegadores web para exibir o conteúdo de forma organizada e visualmente atraente aos usuários.
CSS
CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação e o layout de documentos escritos em HTML ou XML. Com o CSS, é possível controlar a aparência visual dos elementos da página, como cores, fontes, tamanhos, margens e posicionamento. Ele permite separar o conteúdo da estrutura, facilitando a manutenção e a personalização do design das páginas web.
JavaScript
JavaScript é uma linguagem de programação amplamente utilizada no desenvolvimento web para adicionar interatividade e dinamismo às páginas. Com o JavaScript, é possível criar efeitos visuais, validar formulários, manipular elementos da página em tempo real e interagir com servidores por meio de requisições assíncronas. Ele é executado no lado do cliente (navegador) e é uma das principais tecnologias da web, junto com HTML e CSS.
